MES COMPETENCES

Véritable touche à tout depuis ma plus tendre enfance, c’est à l’adolescence que je tombe dans la marmite de l’informatique sans jamais pouvoir en sortir... Et c’est à 23 ans que je découvre le développement par le biais de la maintenance d’un site que l’on m’avait confié.

Depuis, je me forme à cette discipline au gré de mes recherches et expériences, j’ai découvert plusieurs langages :

  • HTML
  • CSS (Sass)
  • PHP
  • MySQL

Mais c'est avec Java que J'ai la révélation !

  • Mini application de gestion de comptes bancaires en ligne de commande
  • Génération de formes géométriques dans une interface graphique
  • Déplacement de formes géométrique avec contraintes dans une interface graphique

Aujourd’hui, à 40 ans, en pleine reconversion, je décide de me professionnaliser dans le développement Java.

Mon appétence pour l’organisation et les bases de données m’attire vers l’informatique de gestion. Le développement d’un ERP serait pour moi un véritable défi de taille.

De plus, ce fort intérêt pour les imposantes infrastructures m’ont permis également de découvrir de nouvelles pratiques que je désirerais mettre en application telles que le TDD (Test Driven Developpment) ou la Clean architecture.
